# Smart-Railway-Signal-Automation

This is a base model of the Smart Railway Signal Automation project.

When a train approaches the signal, the IR sensor detects it and sends a signal to the Arduino. The Arduino changes the light from green to red to stop other trains. After the train passes, the light switches to orange as a warning and then to green, indicating the track is clear automatically.

⚠️ Project Status: In Progress (Prototype / Base Model)
